## How To Run
1. Fork the repo `git clone https://github.com/CremaLabs/crema-api-server.git && cd crema-api-server`
2. Install package dependencies: `npm install`
3. Go: `npm start`

## Dependencies
* Node JS
* npm
* PostgreSQL
